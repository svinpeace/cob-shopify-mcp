## 1. Executive Context

**Product Intent:** The CLI must serve two audiences: humans in terminals (readable tables) and AI agents/scripts (structured JSON). TTY detection auto-switches between modes. `--fields` enables token-efficient output by selecting specific response keys. `--jq` enables inline JSON filtering without external dependencies.

**System Position:** New module at `src/cli/output/` used by every tool command's handler. Sits between `ToolEngine.execute()` result and `process.stdout.write()`.

## 2. Feature Specification

**What Exists Today:**
- `formatTable()` utility at `src/cli/utils.ts:18-31` — basic text table formatting
- `tools run` currently dumps raw `JSON.stringify(result.data, null, 2)` to stdout (`src/cli/commands/tools/run.ts:136`)
- Cost summary printed to stderr via `consola.info` (`src/cli/commands/tools/run.ts:141-145`)

**What Must Be Built:**
1. `OutputFormatter` class/module with methods:
   - `format(data: unknown, options: OutputOptions): string` — main entry point
   - `isTTY(): boolean` — checks `process.stdout.isTTY`
   - `toTable(data: unknown): string` — converts data to aligned table (enhances existing `formatTable`)
   - `toJSON(data: unknown): string` — compact JSON
   - `filterFields(data: unknown, fields: string[]): unknown` — post-execution field selection
   - `applyJq(data: unknown, expr: string): unknown` — jq expression filtering
2. `OutputOptions` type: `{ json?: boolean, fields?: string, jq?: string }`
3. `formatCostSummary(stats: SessionCostStats): string` — stderr cost line
4. `formatError(error: Error | string, code: string): string` — stderr JSON error

**User Workflow:**
```bash
# TTY → table
cob-shopify products list --limit 3

# Piped → JSON auto
cob-shopify products list --limit 3 | jq .

# Explicit JSON
cob-shopify products list --limit 3 --json

# Field selection (implies JSON)
cob-shopify products list --limit 3 --fields id,title,status

# jq filtering (implies JSON)
cob-shopify products list --json --jq '.products[].title'
```

**Risk Assessment:** Low. Pure formatting — no mutations, no API calls. jq dependency choice matters for bundle size.
